# FPGA Blackjack

## Objective

Develop a simplified Blackjack game using Verilog on the Basys 3 FPGA.

The project will emphasize:

- RTL Design
- Finite State Machines
- Verification using Icarus Verilog
- Waveform debugging using GTKWave
- FPGA implementation using Vivado

---

## Features

- Single player
- Dealer AI
- Hit
- Stand
- Reset
- Automatic dealing
- Winner detection

---

## Excluded Features

- Betting
- Insurance
- Splitting
- Double Down
- Multiple players

---

## Development Philosophy

Every hardware module must:

1. Be designed
2. Be simulated
3. Pass verification
4. Be committed to GitHub
5. Then be integrated into the FPGA
